PIXELTONE

about this beautifully unnecessary project

What is PixelTone?

PixelTone is a web-based SSTV (Slow-Scan Television) encoder and decoder, derived from its predecessor PixelPulse - a Java-based college project that started this whole mess.

At some point, the developer decided JavaScript needed more radio noise. PixelTone is the result.

What is SSTV?

SSTV is a method of transmitting images over radio using audio frequencies. Pixels become tones. Tones become pixels again.

It is slow, noisy, inefficient, and deeply satisfying.

How PixelTone Works

Images are converted into audio by mapping pixel intensities to frequency ranges and generating WAV signals.

Decoding reverses the process using signal analysis, frequency detection, and a tolerance for noise.

Why This Exists (No One Asked)

Because radio deserves pixels.

Because modern systems are too clean.

And because doing things the hard way builds character.
