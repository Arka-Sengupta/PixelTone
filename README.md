# PixelTone
A web-based application for encoding images into robot-36 SSTV signals and decoding those SSTV signals back to images
![Logo](https://pixeltone.netlify.app/assets/radio.png)

## Run Locally

Clone the project

```bash
  git clone https://github.com/Arka-Sengupta/PixelTone
```

Go to the project directory

```bash
  cd PixelTone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
